Barc Chat provides a simple yet feature rich chat room for your whole community to communicate in real-time directly on your site.

== Description ==

[Barc](http://barc.com)’s simple tool combines social networking, instant messaging and customer support - without any ads!

##### Current Features:

- Share pictures, videos,and music all within the chat
- Users can live stream their Webcam and microphone (using WebRTC)
- Make rich interactive polls inside the chat
- Chat as a guest or login using popular Social Media
- Create private chats rooms which can be permanently deleted
- Use real-time language translation to chat with people in different languages
- Choose between a ‘floating widget’ or embedded directly in the page
- Purchase and tip Bitcoins to other users directly inside the chat.

Because Barc is hosted in the cloud, you can have unlimited simultaneous users without any special configurations or performance hits to your site. Website owners can unlock moderation control which provides features such as Deleting comments, Banning users, Clearing the chat or Disabling guests from chatting.

Barc is extremely fast, simple to install and is a compelling social solution for any site! Chat with us on Barc.com or Download the plugin to get started!

== Installation ==

Barc takes less than a minute to setup.

- To begin click on the download button, and your download will begin. From there go to your wordpress administration page. Click on Plugins, then click on add new, and select ‘upload.’ Drag the Barc file into the choose upload button - and simply click on install now. Once the installation completes activate the plugin.
